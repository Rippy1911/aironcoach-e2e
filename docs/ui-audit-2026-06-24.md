## P0 — Szczegóły bugów

### UI-05 — Duplicate UserProfile / zły profil — **NAPRAWIONE NA PROD (2026-06-25)**

**Było:** My Profile → `mkpiwecki`, coach save 403.  
**Teraz:** `yse()` + `uu(authUser)` + query `created_by_id`; My Profile → Loki Stream; Hub/Services odblokowane.  
**Uwaga:** funkcja nadal nazywa się `uu`, nie `getCanonicalUserProfile` — to tylko nazwa w bundle.  
**Prompt:** `prompts/base44-prompt-07-duplicate-profile-fix.txt` — **DONE**

### UI-03 — Coach Hub „You're not a coach yet"

**Symptom:** Po „Coach profile created" → pusty hub mimo COACH badge w menu.  
**Root cause:** Gate 1 (`coach_business_name`) OK → Gate 2 `getCoachStats()` fail → UI mapuje na `notCoach`.  
**Prod check (anon):** `POST /functions/getCoachStats` → **401** (endpoint istnieje; błąd 404 „Deployment does not exist" wymaga weryfikacji **zalogowanym** coachem).  
**Prompt:** `prompts/base44-prompt-05-coach-hub-getCoachStats.txt`

### UI-01 — Coach tab ostatni, default Overview — **CZĘŚCIOWO NAPRAWIONE**

**Bundle (2026-06-25):** `coach | overview | activity | follows | stats` — Coach **pierwszy**.  
**Do sprawdzenia:** default tab na publicznym profilu coacha (qa-sweep nadal flaguje Overview first na directory link).  
**Prompt:** `prompts/base44-prompt-11-coach-card-services-layout.txt`

### UI-02 — Services: brak Usuń / Archive w UI — **CZĘŚCIOWO NAPRAWIONE**
