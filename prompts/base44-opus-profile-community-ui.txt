# CZĘŚĆ C0 — ANTI-DUPLIKACJA CTA (P0 — ZRÓB NAJPIERW)

## Problem (potwierdzone na prod, screenshot użytkownika)

Na Feedzie użytkownik widzi TRZY sposoby na to samo:
1. Pomarańczowy przycisk „Create post" w prawym górnym rogu headera
2. Duży inline composer „Share a tip, announcement or update…" w main column
3. „Create a post" w Quick Actions (prawy rail)

To nie ma sensu. Usuń duplikaty.

## Reguła (obowiązuje WSZYSTKIE 4 taby Community)

```
Header CTA = TYLKO gdy main column NIE MA już primary action dla tej samej akcji.
Quick Actions = skróty do INNYCH tabów/routów, NIE duplikat bieżącego taba.
Jedna akcja = jeden primary entry point na ekranie.
```

## Konkretne instrukcje per tab

### Feed
- USUŃ header CTA „Create post" — całkowicie, na desktop i mobile
- USUŃ „Create a post" z Quick Actions rail na Feedzie
- Inline composer u góry feedu = JEDYNY primary „create post"
- ComposeFAB = opcjonalny secondary tylko na mobile (skrót), ukryj gdy composer widoczny
- Header Feed = tylko: tytuł „Community" + taby (Feed/Teams/Messages/People). BEZ subtitle. BEZ CTA.

### Teams
- Header CTA „Create team" = OK (jeśli main column nie ma inline create)
- Quick Actions: NIE dodawaj „Create team" jeśli jest w headerze
- Quick Actions na Teams: Invite member, Browse directory, My trainees — skróty cross-tab

### Messages
- Header CTA „New message" = OK
- Quick Actions: NIE duplikuj „New message"
- Quick Actions na Messages: Go to Feed, Find a coach, Go to teams

### People
- Header CTA „Invite" = OK
- Quick Actions: NIE duplikuj „Invite"
- Quick Actions na People: Discover coaches, Go to Feed, My follows

## Pliki do edycji
- components/community/CommunityHeaderCTA.jsx — zwracaj null dla tab=feed
- components/community/CommunityShell.jsx — nie renderuj headerAction na Feedzie
- components/community/FeedTab.jsx — inline composer = sole primary
- prawy rail / Quick Actions component — per-tab lista BEZ duplikatów (feed: bez create post)

## Acceptance (obowiązkowe)
- Feed: dokładnie JEDEN widoczny entry point „create post" = inline composer
- Screenshot Feed pokazujący: brak „Create post" w headerze, brak w Quick Actions, jest composer
