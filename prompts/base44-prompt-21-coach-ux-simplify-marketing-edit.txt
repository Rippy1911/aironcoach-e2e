3. **Za dużo dróg do tego samego**

## Oczekiwane zachowanie

### Własny profil (coach i zwykły user u siebie)

- **Domyślnie Edit mode** — ołówki / możliwość edycji od razu po wejściu na swój profil.
- **Guest view** = opcjonalny toggle podglądu (jak widzi gość), nie tryb startowy.
- Przy każdym edytowalnym polu: **ikona ołówka** → edycja na miejscu (jak w prompt 20).
- Jeśli pole jest złożone (cała wizytówka coacha): ołówek lub przycisk **„Edytuj w Marketing”** → `/CoachMarketing` z focusem na sekcję.

### Coach Hub — jeden jasny flow edycji

| Akcja na Hub | Dokąd |
|--------------|--------|
| Edit coach profile / Edytuj profil | **CoachMarketing** (nie Settings) |
| Open coach profile | UserProfile + tab Coach (Guest view OFF = edit mode) |
| Preview / Guest view | UserProfile w trybie podglądu gościa |
| Services / oferty | CoachServices |

Hub = **dashboard** (statystyki, plan, skróty). **Edycja treści wizytówki = Marketing.**

### Marketing = główne studio coacha

- CoachMarketing to **jedno miejsce** na: zdjęcie, bio, specjalizacje, business info, widoczność, checklistę kompletności.
- Zmiany z Marketingu **od razu** widać na publicznym profilu (obietnica już w copy — ma działać).
- Settings → Coach Profile: albo link „Otwórz Marketing”, albo tylko lifecycle (pause/deactivate) — **nie duplikuj** pełnego formularza.

### Uproszczenie nawigacji (coach)

User coach powinien rozumieć:
- **Profil** = jak wyglądam (podgląd + szybkie ołówki)
- **Marketing** = pełna edycja wizytówki
- **Hub** = biznes / statystyki
- **Services** = oferty

Usuń lub przekieruj zbędne duplikaty (np. CreateCoachProfile tylko dla **pierwszego** setupu, potem edycja → Marketing).

### Oferty

- Aktywne oferty z Services na tab Coach profilu.
- Ołówek na ofercie → szybka edycja lub „Manage in Services”.

### Przy tworzeniu coacha

- Pytanie o publiczność, **default zaznaczony (publiczny)** — bez zmian.
